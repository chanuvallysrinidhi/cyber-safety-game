
# Cyber Safety Game

An educational web-based game designed to promote cyber safety awareness. The game helps users learn about online security, phishing scams, password protection, digital privacy, and safe internet practices through interactive challenges and engaging gameplay.

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
The project was developed to make cybersecurity concepts easy to understand and accessible through a fun and interactive learning experience.

## Key Features
- Interactive gameplay
- Cybersecurity awareness challenges
- User-friendly interface
- Educational learning experience
- Responsive design
